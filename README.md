# 今天吃什么

一个移动端优先的随机吃饭选择工具，用来快速决定当前时间段该吃什么。项目会根据时间自动切换早餐、正餐和夜宵场景，并把个人设置保存在浏览器本地。

在线地址：https://what-to-eat-six.vercel.app/

## 核心功能

- 按时间自动切换早餐、正餐、夜宵
- 早餐独立卡池
- 正餐支持到店 / 宿舍
- 夜宵使用泡面卡池
- 菜单库四个仓库
- 菜单库快速跳转
- 上划下一条、下划上一条
- 双击常吃
- LocalStorage 本地保存

## 技术栈

- React
- Vite
- Tailwind CSS
- LocalStorage

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
